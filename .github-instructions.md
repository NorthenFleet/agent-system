# GitHub 仓库创建指南

## 方式 1: 使用 GitHub CLI (推荐)

```bash
# 1. 登录 GitHub
gh auth login

# 2. 创建仓库
gh repo create team-dashboard --public --description "OpenClaw 团队信息看板" --source=. --remote=origin

# 3. 推送代码
git push -u origin main
```

## 方式 2: 手动创建

### 步骤 1: 创建 GitHub 仓库

1. 访问 https://github.com/new
2. 仓库名：`team-dashboard`
3. 描述：`OpenClaw 团队信息看板 - Vue 3 + FastAPI`
4. 可见性：Public（公开）或 Private（私有）
5. **不要** 勾选 "Initialize this repository with a README"
6. 点击 "Create repository"

### 步骤 2: 关联远程仓库

```bash
# 替换 YOUR_USERNAME 为你的 GitHub 用户名
git remote add origin https://github.com/YOUR_USERNAME/team-dashboard.git

# 推送代码
git branch -M main
git push -u origin main
```

### 步骤 3: 验证推送

访问 https://github.com/YOUR_USERNAME/team-dashboard 查看代码
