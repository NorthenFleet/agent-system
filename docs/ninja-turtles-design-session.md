### 三、API 设计

```
base_url: http://192.168.31.41:3020/api/v1

# 智能体
GET    /agents              # 获取智能体列表
GET    /agents/{id}         # 获取智能体详情
POST   /agents              # 创建智能体
PUT    /agents/{id}         # 更新智能体
DELETE /agents/{id}         # 删除智能体
POST   /agents/{id}/assign  # 分配任务

# 设备
GET    /devices             # 获取设备列表
GET    /devices/{id}        # 获取设备详情
GET    /devices/stats       # 设备统计

# 任务
GET    /tasks               # 获取任务列表
POST   /tasks               # 创建任务
PUT    /tasks/{id}/status   # 更新任务状态
DELETE /tasks/{id}          # 删除任务

# 团队
GET    /team/status         # 团队状态
GET    /team/stats          # 团队统计

# WebSocket
ws://192.168.31.41:3020/ws/status  # 实时状态推送
```

### 四、数据模型

#### 智能体模型
```python
class Agent(BaseModel):
    id: str
    name: str
    role: str
    status: str  # online, busy, idle, pending
    team: str    # autobots, ninja_turtles
    current_task: Optional[str]
    capabilities: List[str]
    device_id: Optional[str]
    created_at: datetime
    updated_at: datetime
```

#### 设备模型
```python
class Device(BaseModel):
    id: str
    name: str
    ip: str
    os: str
    role: str
    status: str  # online, pending
    ports: List[int]
    specs: dict  # cpu, ram, storage, gpu
    assigned_agents: List[str]
    location: str
    description: str
```

#### 任务模型
```python
class Task(BaseModel):
    id: str
    title: str
    description: str
    status: str  # pending, in_progress, done
    priority: str  # low, medium, high
    assignee: str
    progress: int  # 0-100
    due_date: Optional[datetime]
    created_at: datetime
    updated_at: datetime
```
