## 2. 展示规则

1. 组织看板不再展示任何项目分身，包括已注册的 `wheeljack-*` 和规划中的 `ironhide-*`、`ultra-magnus-*`。
2. 项目经理层只显示 3 个正式项目经理：千斤顶、铁皮、通天晓。
3. 忍者神龟开发组作为开发执行能力池，显示在项目经理下方。
4. 保障层显示在最下方，包含大黄蜂、感知器、救护车、爵士、震荡波、声波。
5. `项目经理`、`忍者神龟开发组`、`保障` 是结构性分组，只用于组织归类，不显示为独立卡片。
