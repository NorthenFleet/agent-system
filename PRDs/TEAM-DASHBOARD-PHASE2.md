# PRD: 团队看板二期 - 统计报表功能

**版本**: 1.0  
**创建日期**: 2026-04-15  
**负责人**: 孙总  
**优先级**: P0

---

## 1. 项目概述

### 1.1 背景
当前团队看板缺少统计报表功能，无法直观展示团队工作效率和项目进度。需要增加统计报表模块，帮助孙总快速了解团队状态。

### 1.2 目标
- 实现团队工作效率统计
- 实现项目进度可视化
- 支持报表导出功能

### 1.3 范围
**包含**:
- 工作效率统计图表
- 项目进度甘特图
- 报表导出 (PDF/Excel)

**不包含**:
- 移动端适配
- 实时数据同步

---

## 2. 功能需求

### 2.1 功能列表

| 编号 | 功能名称 | 描述 | 优先级 | 预计工时 |
|------|----------|------|--------|----------|
| F001 | 效率统计图表 | 展示团队工作效率趋势 | P0 | 8h |
| F002 | 项目进度甘特图 | 可视化项目进度 | P0 | 8h |
| F003 | 报表导出 | 支持 PDF/Excel 导出 | P1 | 4h |

### 2.2 功能详情

#### F001: 效率统计图表

**用户故事**:
> 作为孙总，我希望查看团队工作效率趋势，以便了解团队工作状态。

**验收标准**:
- [ ] 显示近 7 天效率趋势
- [ ] 显示近 30 天效率趋势
- [ ] 支持按智能体筛选

**技术要点**:
- 技术栈：ECharts + Vue 3
- 接口：GET /api/v1/stats/efficiency
- 数据：从任务完成记录聚合

#### F002: 项目进度甘特图

**用户故事**:
> 作为孙总，我希望查看项目进度甘特图，以便了解各项目进展。

**验收标准**:
- [ ] 显示所有项目进行中任务
- [ ] 支持任务拖拽调整
- [ ] 显示里程碑标记

**技术要点**:
- 技术栈：dhtmlxGantt + Vue 3
- 接口：GET /api/v1/projects/{id}/timeline
- 数据：从任务表关联项目表

#### F003: 报表导出

**用户故事**:
> 作为孙总，我希望导出统计报表，以便存档和分享。

**验收标准**:
- [ ] 支持 PDF 格式导出
- [ ] 支持 Excel 格式导出
- [ ] 导出包含所有统计数据

**技术要点**:
- 技术栈：pdfkit + openpyxl
- 接口：POST /api/v1/stats/export
- 数据：聚合统计结果

## 4. 技术架构

### 4.1 系统架构图
```
前端 (Vue 3 + ECharts)
    ↓
后端 API (FastAPI)
    ↓
数据库 (SQLite)
```

### 4.2 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | Vue 3 + ECharts | 图表库 |
| 后端 | FastAPI | API 服务 |
| 数据库 | SQLite | 数据存储 |
| 导出 | pdfkit + openpyxl | 报表生成 |

### 4.3 数据模型

```sql
-- 效率统计表
CREATE TABLE efficiency_stats (
    id INTEGER PRIMARY KEY,
    agent_id TEXT NOT NULL,
    date DATE NOT NULL,
    completed_tasks INTEGER,
    efficiency_score REAL
);

-- 项目进度表
CREATE TABLE project_timeline (
    id INTEGER PRIMARY KEY,
    project_id TEXT NOT NULL,
    task_id TEXT NOT NULL,
    start_date DATE,
    end_date DATE,
    progress REAL
);
```
