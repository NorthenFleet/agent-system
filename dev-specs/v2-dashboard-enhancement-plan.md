## 三、功能模块

### 模块 A: 任务管理增强 (🟥 拉斐尔 + 🟪 多纳泰罗)

#### 后端 (拉斐尔)
- [ ] A1. **任务 CRUD API** — 创建/读取/更新/删除任务
- [ ] A2. **任务分配** — 指派给 Agent + 变更历史
- [ ] A3. **进度追踪** — 子任务进度 → 自动汇总到主任务
- [ ] A4. **任务筛选/搜索** — 按状态/优先级/负责人/标签过滤
- [ ] A5. **任务评论** — 任务级讨论/备注
- [ ] A6. **任务模板** — 常用任务快速创建
- [ ] A7. **数据库迁移** — 从文件存储迁移到 PostgreSQL

#### 前端 (多纳泰罗)
- [ ] A8. **任务列表增强** — 表格视图 + 排序 + 分页
- [ ] A9. **任务详情面板** — 右侧滑出，显示完整信息
- [ ] A10. **任务创建/编辑对话框** — 表单 + 验证
- [ ] A11. **看板视图** — 拖拽式 Kanban 列 (待办/进行中/完成)
- [ ] A12. **甘特图视图** — 任务时间线 (ECharts)

### 模块 B: Agent 实时监控 (🟥 拉斐尔 + 🟪 多纳泰罗)

#### 后端 (拉斐尔)
- [ ] B1. **心跳 API** — Agent 定期上报状态 (`POST /api/agents/{id}/heartbeat`)
- [ ] B2. **状态采集器** — 定时调用 `openclaw status` 获取真实状态
- [ ] B3. **任务绑定** — Agent 当前任务关联到任务系统
- [ ] B4. **历史日志** — Agent 状态变更历史 + 持续时间统计
- [ ] B5. **告警规则** — Agent 离线超 N 分钟 → 告警
- [ ] B6. **WebSocket 推送** — 实时状态变更推送到前端

#### 前端 (多纳泰罗)
- [ ] B7. **Agent 状态卡片** — 实时在线/离线/忙碌 + 当前任务
- [ ] B8. **心跳指示灯** — 绿色在线 / 黄色超时 / 红色离线
- [ ] B9. **Agent 详情面板** — 点击展开，显示任务历史 + 统计
- [ ] B10. **实时通知** — Agent 状态变更 → 右上角通知

### 模块 C: 团队访问 & 登录 (🟥 拉斐尔 + 🟦 李奥纳多)

#### 后端 (拉斐尔)
- [ ] C1. **用户认证 API** — 登录/登出/Token 刷新
- [ ] C2. **角色权限** — admin / viewer / agent (不同权限范围)
- [ ] C3. **Session 管理** — JWT Token + Cookie
- [ ] C4. **用户管理** — 创建/禁用用户 (admin 专用)

#### 前端 (多纳泰罗)
- [ ] C5. **登录页面** — 用户名/密码登录
- [ ] C6. **权限路由守卫** — 未登录跳转登录页
- [ ] C7. **用户菜单** — 右上角头像 + 登出

#### 架构 (李奥纳多)
- [ ] C8. **认证方案设计** — JWT vs Session 选型 + 安全规范
